## 二、订阅体系

### 1. 银星卡 (Silver Pass) - 留存向

对标：原神"空月祝福"

**价格**：$4.99/月

**权益**：
- 购买即得：300 月石 (价值 $3)
- 每日领取：50 月石 × 30 天 = 1,500 月石
- **总价值**：1,800 月石 (300%+ 返利)

**用户心理**：太划算了，必须每天上线领 → 极大提高 DAU

**实现优先级**：后期

---

### 2. 金星特权 (Gold Membership) - 体验向

对标：原神"珍珠纪行" + ChatGPT Plus

**价格**：$9.99/月 (或 $14.99)

**权益**：
- 解锁角色：Vesper (魅魔/反派) 永久解锁
- 无限体力：每日聊天无限制
- 双倍经验：聊天获得的 XP 翻倍
- 专属铭牌：聊天气泡变为金色

**用户心理**：我是尊贵的 VIP，可以玩最刺激的角色

**实现优先级**：MVP 保留基础订阅框架，功能后期补全

## 三、礼物系统

### 设计原则

1. **消除付费羞耻感**：不是"花钱买色图"，而是"送礼物哄女朋友"
2. **数值平衡**：免费党 60 天、小月卡 20 天、氪金当天达 Lv.20
3. **无限上限**：持续推出新礼物，榜单排名永远有追求空间

### XP 计算公式

```
基础: 1 月石 = 1 XP
高级礼物: 价格 × XP倍率
```

---

## 四、MVP 配置

### 上线必须包含

1. **3 种基础礼物**：咖啡 ☕、玫瑰 🌹、蛋糕 🍰
2. **2 种状态礼物**：红酒 🍷、悔过书 📜 → 验证用户是否愿意为 Prompt 改变付费
3. **1 种大额礼物**：钻戒 💍 ($20) → 验证土豪冲动消费能力

### 暂不实现

- 月卡系统 (银星卡)
- 充值档位货币兑换 (先做 IAP 直接购买礼物)
- 全服广播
- 照片生成礼物

## 五、收益分析

### 成本估算

- Token 成本（优化后模型）：~$0.002/条
- 免费用户 (30条/天)：$0.06/天 ≈ $1.8/月
- 付费用户 (100条/天)：$0.2/天 ≈ $6/月

### 收益模型

| 用户类型 | 月收入 | 成本 | 利润 |
|----------|--------|------|------|
| 银星卡 $4.99 | $4.99 | ~$6 | 微利/亏本，但转化率高 |
| 金星卡 $9.99 | $9.99 | ~$6 | ~$4/人/月 稳定现金流 |
| 大R充值 $99.99 | $99.99 | ~$6 | **~$94** 一个土豪养活 50 个免费用户 |

## 六、技术实现要点

### 状态礼物 (Tier 2) Prompt 注入

```python
# 红酒效果
{
    "status": "tipsy",
    "prompt_modifier": "你现在有点微醺，说话变得更加柔软和放松，防御心降低，更容易说出平时不会说的话...",
    "duration_messages": 30,
    "started_at": "2026-02-01T10:00:00Z"
}
```

### 效果持续追踪

```python
class ActiveEffect:
    effect_type: str       # "tipsy", "maid_mode", "truth_serum"
    remaining_messages: int
    prompt_modifier: str
    started_at: datetime
```

每次对话前检查 `remaining_messages`，注入对应 prompt_modifier。
